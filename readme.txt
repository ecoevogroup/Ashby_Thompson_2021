readme.txt for "Non-pharmaceutical interventions and the emergence of pathogen variants"

All files created and tested with MATLAB R2019b

Files:

variant_evo.c - source code for running individual simulations. Compile in Matlab with "mex variant_evo.c"
variantevo_sweep.m - carries out the main parameter sweep for figures 2 and 5
variantevo_mortality_sweep.m - carries out the main parameter sweep for figures 4 and S1
fig_gen.m - figure generation file (all figures)
figs_2_5.m - generates figures 2 and 5
figs_summary.m - generates summary data for figures 2, 5, S1-2, S4-5
fig_3_data.m - generates data for figure 3
fig_3.m - generates figure 3
figs_4_6.m - generates figures 4 and 6
figs_S1_S2_S4_S5.m - generates figures S1, S2, S4 and S5
figs_S3_S6.m - generates figures S3 and S6
Data/...mat - data files for figures
